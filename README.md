# CycleRent

Cycle rental platform — book a cycle, ride anywhere.

## Project structure


## Running locally

### Backend
```bash
cd cycletwo
npm install
npx prisma db push
npx prisma generate
npm run dev
```
Runs on http://localhost:5000

### Frontend
```bash
cd cyclefront
npm install
npm run dev
```
Runs on http://localhost:3000
